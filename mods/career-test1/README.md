# 职业模组（测试1）

增加一个职业，该职业包含两个职业特性

订阅该模组后，可以在游戏的“选择职业”页面的最下方，看到新增的职业


该模组依赖下面的职业特性模组：
- [career-feature-increasedamagebycoin](https://github.com/yyc-git/GiantessPlay-Mod/tree/main/mods/career-feature-increasedamagebycoin)
- [career-feature-increasefullhp](https://github.com/yyc-git/GiantessPlay-Mod/tree/main/mods/career-feature-increasefullhp)


# Career Mod(Test 1)

Add a career that includes two career features

After subscribing to this mod, you can see the newly added career at the bottom of the "Select Career" page in the game.

This mod depends on the following career feature mods:
- [career-feature-increasedamagebycoin](https://github.com/yyc-git/GiantessPlay-Mod/tree/main/mods/career-feature-increasedamagebycoin)
- [career-feature-increasefullhp](https://github.com/yyc-git/GiantessPlay-Mod/tree/main/mods/career-feature-increasefullhp)
