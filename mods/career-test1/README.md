# 中文

## 描述

增加一个职业，该职业包含两个职业特性

订阅该模组后，可以在游戏的“选择职业”页面的最上方，看到新增的职业


## 特性
- 生命上限增加100%
- 每持有50金币，伤害增加1%


# English

## Description

Add a career that includes two career features

After subscribing to this mod, you can see the newly added career at the top of the "Select Career" page in the game.


## Features
- Max HP increased by 100%
- For every 50 gold coins held, damage increases by 1%
